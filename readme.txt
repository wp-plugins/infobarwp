== Description ==

InfobarWP is a notification bar plugin which allows you to:

* Display your notification bars in website header or footer
* Customize with custom background and text clots
* Customize the notification bar text, link and background color
* 5 built-in notification bar color schemes
* Fully responsive and support for mobile and tablets
* Target specific pages, posts, categories (or exclude them)
* Run your campaign on specific days, weeks or all the time
* Only display infobars to your visitors after “X” number of page views

For more advanced features including split testing and analytics, please refer to [InfobarWP Pro](http://infobarwp.com/ "Notification Bar Plugin").

== Installation ==

1. Upload plugin files to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
